# Red Hat Certified OpenShift Administrator Exam Preparation Course

This course is curated for learning all the necessary skill required to pass Red Hat Certified OpenShift Administrator Exam.

## Prerequisites

- Linux Administration
- Command Line proficiency
- Basic Networking knowledge

- Basic understanding of Containers (Docker), Container Lifecycle Management
- Kubernetes Basics
